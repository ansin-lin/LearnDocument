# 第1章 Linux 基础入门

Linux 是一种开源操作系统，广泛应用于服务器、云计算与嵌入式系统。

## 1.1 什么是 Linux

- 开源、免费、高安全性、稳定性强。
- 常见发行版：Ubuntu、CentOS、Rocky Linux、Debian。

## 1.2 Linux 与 Windows 区别

| 对比项 | Linux | Windows |
|--------|--------|----------|
| 文件系统 | 区分大小写 | 不区分 |
| 权限机制 | 严格 | 宽松 |
| 软件安装 | apt/yum | exe |
| 用途 | 服务器/云计算 | 桌面系统 |

## 1.3 目录结构

| 目录 | 功能 |
|------|------|
| / | 根目录 |
| /bin | 命令 |
| /etc | 配置文件 |
| /home | 用户主目录 |
| /var | 日志与动态文件 |
| /usr | 用户程序 |

## 练习题

1. 使用命令列出 /usr 目录内容。
2. 查看当前路径并进入 /etc 目录。

一、阶段概览
阶段	学习周期	学习目标
第1阶段：入门与环境搭建	1周	理解Spring Boot是什么，会搭建运行第一个项目
第2阶段：核心注解与配置	2周	掌握依赖注入、配置文件、控制层基础
第3阶段：Web开发核心	3周	掌握Controller、Restful接口、参数传递、模板引擎
第4阶段：数据库操作	3周	学会用Spring Data JPA/MyBatis连接数据库
第5阶段：综合项目实战	2~3周	做出一个小型管理系统（如用户管理/博客）
第6阶段：进阶与部署	2周	掌握打包部署、配置文件分环境、异常处理、拦截器等
🧩 二、详细学习路线
✅ 阶段1：Spring Boot 入门

目标：了解Spring体系、创建第一个Spring Boot项目

知识点：

Spring、Spring MVC、Spring Boot之间的关系

什么是IOC、DI、AOP（只需概念）

使用 IDEA 创建第一个 Spring Boot 项目

启动类结构（@SpringBootApplication）

运行方式（IDEA、命令行、打包Jar）

练习任务：

创建一个能返回 "Hello Spring Boot" 的接口

改一下端口号、项目名，理解配置文件作用

✅ 阶段2：核心注解与配置

目标：理解 Spring Boot 的核心机制：IOC、配置、注解驱动

知识点：

@Component、@Service、@Repository、@Controller 的区别

@Autowired 与依赖注入

@Configuration、@Bean 自定义配置

application.properties / application.yml

多环境配置（dev/test/prod）

练习任务：

定义一个 Service 类，注入到 Controller 中调用

学会从 yml 文件中读取配置参数（@Value, @ConfigurationProperties）

✅ 阶段3：Web 开发与 Restful 接口

目标：掌握后端接口开发流程

知识点：

@RestController 与 @RequestMapping

GET / POST / PUT / DELETE 的使用

@RequestParam、@PathVariable、@RequestBody

JSON 数据返回（Jackson）

异常处理（@ExceptionHandler, @ControllerAdvice）

模板引擎 Thymeleaf（可选）

练习任务：

编写一个用户注册+登录的接口

用 Postman 调试API，理解请求体与响应体

✅ 阶段4：数据库与持久层

目标：让项目能操作真实数据库

知识点：

数据库连接配置（application.yml）

Spring Data JPA 与 MyBatis 二选一（推荐都学）

Repository / Mapper 概念

实体类映射（@Entity, @Table, @Column）

MyBatis XML 映射文件写法、注解写法

事务管理（@Transactional）

练习任务：

连接 MySQL，创建 user 表

实现 用户增删改查 接口

学会分页查询、条件查询

✅ 阶段5：综合项目实战

目标：整合所学知识，开发一个小系统

推荐项目示例：

用户管理系统

图书管理系统

博客/留言板系统

要点：

前后端接口规划（/user/list、/user/add）

全局异常处理

日志（logback配置）

DTO + VO 结构优化

✅ 阶段6：进阶与部署

目标：能独立上线项目

知识点：

打包部署（mvn clean package → java -jar xxx.jar）

分环境配置（application-dev.yml / application-prod.yml）

自定义拦截器 / 过滤器

统一返回体封装（Result<T>）

定时任务（@Scheduled）

Swagger / Knife4j 接口文档

练习任务：

将项目部署到本地服务器（或AWS EC2）

写一份部署文档
